Follow through these instructions to execute the project's jar file (jar file lies in dist/ folder) :

1. Open the root directory.

## DO NOT Navigate to the dist/ directory

2. Next open the Terminal / Command Line at this location and execute the following command :

> java -jar dist/OOMProject.jar

NOTE: Make sure the project is downloaded as whole and not the jar file alone.


JAVA REQUIREMENTS:
- java 15.0.1

If using Netbeans, clean and build before running. Use jar file preferably.
